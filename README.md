# app-site-resource

## 简介

`app-site-resource` 是一个用于归档和发布多个独立 HTML 页面的公共仓库。  
本仓库不针对任何特定域名或网站，仅作为静态资源的存放与分发点。

仓库中的页面均为独立、自包含的 HTML 文件，可直接通过浏览器打开查看，无需额外服务端支持。

---

## 目录结构

```
app-site-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件代表一个独立的页面。
- `assets/` 目录按需组织，可包含子目录以区分不同页面的资源。
- `docs/` 目录用于存放补充说明、使用指南等。

---

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端接口或数据库。
- 页面内容、样式、脚本均内联或通过相对路径引用 `assets/` 中的资源。
- 文件命名建议使用有意义的小写英文，避免特殊字符。
- 归档时请确保页面在本地测试无误后再提交。

---

## 维护说明

- 欢迎提交新页面或更新已有页面。
- 提交前请检查 HTML 结构完整性，避免引用外部不可达资源。
- 请勿在页面中添加任何形式的跟踪代码、广告或诱导性内容。
- 仓库维护者保留对不合规内容的修改或删除权利。

---

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 下的 HTML 文件查看效果。
3. 也可将仓库部署至任何静态托管服务，实现在线访问。
